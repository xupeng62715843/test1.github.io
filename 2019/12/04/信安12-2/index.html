###什么是计算机网络
将位于不同位置的计算机，用不同的传输介质（有线、无线）连接起来，通过一些网络软件，最终实现资源共享的系统。

思科pacet Tracer
路由器
2960双层交换机
3560三层交换机
Hub 集线器
无线设备
线缆{蓝色：配置线；黑色实线、黑色虚线都是双绞线，实线是直连线，虚线是交叉线；闪电形的两个是串口线}
终端设备{台式机、笔记本、服务器等}

e0/0 以太网接口 传输速率10MB
f0/0 快速以太网接口 传输速率100MB
g0/0 千兆以太网接口 传输速率1GB

思科路由器启动分为：1.硬件检查 2.运行IOS 3.导入配置文件
BIOS：硬件基础系统
思科路由器主要存储硬件{
ROM：只读存储器
Flash:存储完整的IOS镜像文件
NVRAM:存放路由器启动配置文件
RAM：存储运行配置文件
}

双绞线：直通线、交叉线、控制线
T568B： 橙白 橙 绿白 蓝 蓝白 绿 棕白 棕
T568A： 绿白 绿 橙白 蓝 蓝白 橙 棕白 棕

直通线:T568A-T568A T568B-T568B
不同设备相连
交换机-路由器 路由器-集线器 交换机-PC

交叉线：T568A-T568B
交换机-交换器 路由器-路由器 交换机-集线器

全反线：控制线

基本命令操作：

如何对设备进行配置？
真实设备的配置：
需要通过控制线进行连接，并在计算机上安装接口驱动程序和超级终端程序。

模式：
Router>用户模式
Router>enable (简写en)
Router#特权模式
Router#config terminal 全局配置模式
Router(config)#
全局模式下路由器改名：
Router(config)# hostname R1

路由器工作在网络模型的第三层（所以也叫三层【网络层】设备）
网络层：传输数据包，作用是寻址，隔离广播，转发等。
广播域：不同网络处于不同广播域。 路由器的一个接口为一个广播域，交换机的所有接口都在一个广播域。
冲突域：集线器的所有端口都工作在一个冲突域，交换机的一个接口为一个冲突域。

R1(config)# interface fastEthernet 0/0
进入0/0接口
R1(config-if)# 接口模式
R1(config-if)#ip address 192.168.10.254 255.255.255.0
配置ip地址（和子网掩码）

子网掩码：标识网络号
R1(config-if)#no shudown 开启端口

返回上一级模式 exit
直接返回到特权模式 end

网关:一个网络的路由器 填路由器在这个网络接口的ip地址

在全局模式下输入：no ip domain-lookup (取消域名搜索)
ctrl+shift+6

如何查看配置是否正确：
启动配置文件：配置好以后进行保存的配置文件。
运行配置文件：当前已配置但是还未保存或当前运行的配置文件。

在特权模式下，show running-config 查看运行配置文件
show startup-config 查看启动配置文件

特权模式下 copy running-config startup-config 保存配置文件
（或 R1#write R1#w 也可以直接保存）

R1#reload 重启
重启后running-config的内容会消失，而startup-config的配置会一直存在。

###为提高设备安全，配置设备console密码保护设备。(路由器开机密码)
进入全局模式，R1#config t
R1#(config)#line console 0
R1#(config-line)#password 123456
R1#(config-line)#login

####如果需要用户名和密码同时存在，怎么配置呢？
R1#config t
R1#(config)#username xupeng password 123456
R1#(config)#line console 0
R1#(config)#login local

####设置enable密码：
R1#config t
R1(config)# enable password 123456
明文密码
R1(config)# enable secert 123456
密文密码
当这两个密码同时存在时，密文密码生效。

###为方便管理，配置telnet远程管理设备。
R1（config）#line vty
R1（config）#line vty 0 4
R1（config-line）#password 123456
R1（config-line）#login

####如何选择登陆呢？
在pc端的命令提示符中输入：pc>telnet 192.168.10.254

####在远程登陆时需要用户名和密码呢?
R1（config）#line vty 0 4
R1（config-line）#login local

####如何配置管理交换机设备呢？
默认情况下，所有端口都在vlan1这个网络，我们只需要给vlan1这个接口配置IP地址，并远程登陆密码，就可以进行远程登陆访问了。

switch(config)#int vlan1
switch(config-if)#ip address 192.168.10.253 255.255.255.0
switch(config-if)#no shutdown 0

switch(config)#line vty 0 4
switch(config-line)#login local

switch(config)#hostname S1
S1(config)#username xupeng password 123456
S1(config)#enable secret 123456

SSH远程管理网络设备

switch(config)#hostname S1
S1(config)#ip domain-name xupeng.com 设置域名
S1(config)#crypto key generate rsa 配置加密方式为RSA
How many bits in the modulus [512]:1024 加密的位数（默认是512位）
S1(config)#username admin privilege 0 secret xupeng 设置用户名及密码
安全级别是0
s1(config)#line vty 0 4 进入line vty 模式 开启5个端口
S1(config-line)#exec-timeout 10 0 设置超时时间 10分钟 0秒
S1(config-line)#logging synchronous 日志同步
S1(config-line)#login local 采用本地登陆
S1(config-line)#transport input ssh 启用ssh登陆

在pc机上验证ssh登：
ssh -l 用户名 ip地址

####ssh远程访问配置
服务端：
1.创建账户和密码
2.规定远程连接的方式

R1(config)#username admin password 123456

R1(config)#line vty ?
R1(config)#line vty 0 5 （创建了6个）

R1(config-line)#login local

R1(config-line)#transport input ssh
